# geo-3d-earth
A online 3D graphic demo using Three.js and WebGL.

This project includes both high-level and low-level 3D implementations:

- The main scene is built with Three.js for rapid development and interaction.
- A separate module demonstrates low-level WebGL rendering, including custom shaders and matrix transformations.

This reflects my understanding of both practical application development and underlying graphics pipeline concepts.

## Tech-Stack

- WebGL (low-level rendering)
- Three.js (high-level 3D abstraction)
- GLSL (shader programming)
- GeoJSON (geospatial data)
